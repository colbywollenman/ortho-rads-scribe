Google Cloud Setup Guide

Follow these steps once — takes about 5 minutes.

01 — Go to console.cloud.google.com

Create a new project (any name, e.g. "OrthoScribe")

02 — Enable the Google Sheets API

APIs & Services → Enable APIs → search "Google Sheets API" → Enable

03 — Create an OAuth 2.0 Client ID

APIs & Services → Credentials → Create Credentials → OAuth Client ID
Application type: Web application
Authorised JavaScript origins: file:// and your local file path

04 — Copy your Client ID

It looks like: 123456789-abc.apps.googleusercontent.com

05 — Get your Spreadsheet ID

Open your Google Sheet. Copy the long string from the URL:
docs.google.com/spreadsheets/d/THIS_IS_THE_ID/edit

06 — Paste both into the form and click Connect.

Your credentials stay in this browser tab only. Nothing is sent to any server except the Anthropic API (for parsing) and Google Sheets API (for writing).
